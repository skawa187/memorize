# Memorize (dev)

Simple django application to save your thoughts.

## Prerequsites

  

Create two files (secrets) in the `django-app/` directory:
 
- django_sec_dev.json
```
{
    "secret_key": "<secret_key_value>"
}
```
***
- db_sec_dev.json
```
{
    "postgres_password": "<db_password>"
}
```
***
